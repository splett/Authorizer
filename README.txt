This project was carried out in Java1.8 and uses Maven for dependency management.
Uses external libraries:
 - GSon, for reading the JSON informed by the user
 - JUnit, for unit testing

To run the project in Eclipse, follow these steps:
 - If you don't already have it, download Eclipse Oxygen.3a Release (4.7.3a) at https://www.eclipse.org
 - Open Eclise
 - Download the file in a local folder and unzip it in the workspace folder
 - In the Package Explorer tab, right-click and select "Import"
 - Select the option "Existing Projects into Workspace"
 - Select the project folder and finish
 - Update Maven's dependencies if it hasn't downloaded automatically
 - To run the application select the project, right click, select "Run as -> Java Aplication" and finally click on the "Main" class
 - To run the unit tests select the project, right click on "Run as -> JUnit Test"

It is also possible to perform the test using CMD.
It is available in the JAR folder. To run it is only necessary to click on the file "RunAuthorizer.bat"!

Documentation is available in the doc folder. To view click index.hmtl and open it with the browser.
